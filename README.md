# [GeoPing](https://dunkyoreo.github.io/GeoPing/)

GeoPing is an interactive geospatial tool and high-performance 3D weather visualizer that abandons traditional flat maps for deep-dive planetary rendering. 

Wrapped in a minimalist cyberpunk web UI, this tactical data dashboard allows users to scan a Three.js globe, drop pinpoint targeting vectors, and decrypt real-time atmospheric metrics from any coordinate on Earth. 

By mapping a procedurally generated D3.js map texture onto a transparent 3D core, the interface acts as a high-tech reconnaissance scanner for global weather patterns. 





## References
* **Weather Data:** Real-time metrics provided by the open-source [Open-Meteo API](https://open-meteo.com/).
* **Geolocation:** Reverse coordinate mapping provided by the [BigDataCloud API](https://www.bigdatacloud.com/).
* **Geospatial Outlines:** Base country coordinate matrices sourced from open GeoJSON datasets.
